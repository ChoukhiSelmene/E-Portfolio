Jean's Room

Jean's Room est un jeu développé en iJava qui est une version simplifiée de java fournie et maintenue par Yann Secq, enseignant à l'IUT A de l'université de Lille.

Ce jeu est le produit d'un projet ayant pour but de proposer à des enseignants fictifs une solution ludo-pédagogique pour apprendre des notions de culture générale à leurs élèves de niveau primaire.

Jean's Room est donc un 'escape game' en ASCII art se jouant en ligne de commande. Il propose divers énigmes nécessitant la recherche d'informations liées à la culture générale pour trouver le code du coffre contenant la clé de la pièce.

Durant son développement, plusieurs entretiens avec les enseignants fictifs ont eu lieu pour faire des démos et voir l'avancement de la solution. Ce projet étant le premier réalisé en BUT, il nous a appris beaucoup :

Cyber Break

Cyber Break est le nom donné à une escape game virtuelle développée sur le site genial.ly.
Le but de ce projet était de nous faire travailler la gestion d'équipe et le respect des délais mais aussi un moyen ludo-pédagogique pour faire découvrir ou redécourvir le réseau aux étudiants en informatique.

Par groupe de trois, on a dû réaliser s'organiser sur un Trello, penser à un diagramme de Gantt, et fixer des tâches en un temps donné. Aussi, nous devions se répartir les tâches équitablement et savoir gérer les retards sur le calendrier.

API Pizzaland

Une pizzeria nous a demandé de développer une API REST.

Cette API permet de renseigner les pizzas, les ingrédients, et d'autres informations complémentaires de la pizzeria cliente.
Pour la réalisation de cette API, nous avons également mis en place une base de données sous Postgresql.
Le développement de cette API s'est fait en Java J2EE et a été hébergé par le serveur web statique Tomcat.
